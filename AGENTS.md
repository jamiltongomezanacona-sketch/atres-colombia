# AtresColombia

- El proyecto se llama AtresColombia.
- Diseñar primero para celular.
- Mantener una experiencia simple inspirada en WhatsApp Business.
- La primera versión contiene una sola tienda.
- La arquitectura debe admitir múltiples talleres posteriormente.
- Usar Next.js App Router, TypeScript y Tailwind CSS.
- No duplicar componentes.
- No eliminar funciones existentes sin autorización.
- Mantener los textos en español.
- Ejecutar lint y build después de cambios importantes.
- Explicar siempre qué archivos se modificaron.
